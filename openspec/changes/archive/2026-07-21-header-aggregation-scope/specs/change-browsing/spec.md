## REMOVED Requirements

### Requirement: Aggregation toggle control

**Reason**: The aggregation control moves off the Changes page into the global application header for
**both** hosts, now specified by the `aggregation-scope-control` capability (a three-state scope
selector rather than a Changes-page toggle). The Web tri-state and the VS-Code-only aggregate checkbox
are both removed from the change list. The change-row worktree source indicators and worktree-qualified
change links remain in `change-browsing` and are unaffected.
